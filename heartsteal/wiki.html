一个充满挑战与刺激的生存服务器。在这里，每一次击杀都至关重要。

新手入门指南

刚加入服务器？别担心，按照这些步骤开始你的冒险吧！

第一步：生存基础

像正常的生存模式一样，先撸树，制作基础工具。你的首要目标是找到食物并远离夜晚的怪物。我们强烈推荐您使用位于出生点右侧的NPC进行快速传送。您也可以使用/rtp

第二步：寻找安全区

主城是绝对安全的区域。但你需要在野外建立自己的家。使用 /towny map 查看地图，寻找远离其他城镇的区域来避免早期冲突。

第三步：建立你的城镇

一旦你攒够了建立城镇所需的金钱，就找一个心仪的地方输入 /town new [城镇名]。这将保护你的第一个地块，你的家园就有了保障！

第四步：开始赚钱

通过挖矿、耕种或狩猎收集资源，然后使用 /sellgui 或 /sell hand 将它们出售给系统商店来赚取第一桶金。金钱是扩张城镇和购买装备的基础。

服务器规则

为了保证所有玩家的良好体验，请遵守以下规则。违反规则将受到惩罚。

  • 尊重他人: 禁止任何形式的歧视、骚扰、人身攻击或仇恨言论。
  • 禁止作弊: 严禁使用任何作弊客户端、外挂、透视、飞行等不正当手段获取优势。
  • 禁止利用BUG: 禁止恶意利用服务器或游戏BUG。如发现BUG请及时向管理员报告。
  • 文明交流: 禁止在公共频道刷屏、发布广告或不当内容。

核心玩法: 偷心

本服务器的核心机制是“偷心”。生命值是你最宝贵的资源。

击杀与偷心

每次成功击杀一名玩家，你将会从他身上偷取一颗心（即2点生命值上限）。被击杀的玩家将永久失去这颗心。

生命耗尽

当你的生命值上限降至零时，你将被服务器永久封禁。请珍惜你的每一颗心，谨慎行事。

如何获得心？

除了击杀玩家，你还可以通过以下方式恢复或增加你的生命值上限：

  • 合成生命核心: 通过收集稀有材料可以合成生命核心，使用后可增加生命上限。
  • 参与服务器活动: 管理员会不定期举办活动，获胜者可以获得生命奖励。
  • 从其他玩家处购买: 你可以在玩家商店中寻找出售生命核心的商人。
  • 击杀其他玩家：你可以通过击杀其他玩家来获得生命核心。

Towny 指南

Towny可以让你建立和管理自己的城镇，保护你的财产，并与朋友一起发展。

常用指令

  • /town new [城镇名] - 创建新城镇
  • /town claim - 占领脚下地块
  • /town unclaim - 取消占领地块
  • /town add [玩家ID] - 邀请玩家
  • /town kick [玩家ID] - 踢出居民
  • /town deposit [金额] - 向城镇银行存款
  • /town set spawn - 设置城镇传送点
  • /plot fs [价格] - 出售你站立的地块
  • /plot nfs - 取消出售地块
  • /plot claim - 购买待售地块
  • /res set perm ... - 设置地块权限
  • /nation new [国家名] - 建立国家

市长小贴士

  • 税收: 使用 /town set taxes [金额] 来设置每日税收，这是维持城镇运营和扩张的重要资金来源。
  • 地块类型: 使用 /plot set [类型] (例如 arena, embassy, shop) 来设置特殊地块，给予它们不同的功能。
  • 保持活跃: 长期不在线的市长可能会导致城镇变为废墟。请确保定期登录或设置副市长。

经济系统指南

学习如何在服务器中赚钱和交易，成为商业巨头！

QuickShop - 玩家商店

你可以创建自己的箱子商店来向其他玩家出售或收购物品。

  1. 创建出售商店: 放置一个箱子，将要出售的物品放进去。然后手持该物品，左键点击箱子，并在聊天框中输入你希望的单个物品售价。
  2. 创建收购商店: 放置一个箱子（确保里面有足够的钱支付）。手持你想要收购的物品，左键点击箱子，然后在聊天框中输入 /qs buy，再输入你愿意支付的单个物品收购价。
  3. 进行交易: 找到其他玩家的商店，左键点击商店牌子即可按标价购买，右键点击即可按标价出售。

快速出售

快速将你背包里的物品卖给系统换取金钱。

  • /sell hand - 出售你手中拿着的全部同类物品。
  • /sellgui - 打开一个图形界面，将背包里任何可出售的物品拖入其中即可一次性出售。更方便，但需要你手动操作。

/shop - 系统商店

使用 /shop 指令可以打开系统商店 (由Nascraft插件提供)。在这里你可以购买服务器提供的基础物品（如食物、方块），或将你的物品出售给系统（通常价格低于玩家交易）。

如何加入服务器

只需几个简单步骤，即可开始你的偷心之旅！

  1. 第一步: 启动你的Minecraft Java版客户端。我们推荐使用 1.21+ 版本以获得最佳体验。
  2. 第二步: 在主菜单点击“多人游戏 (Multiplayer)”。
  3. 第三步: 点击“添加服务器 (Add Server)”按钮。
  4. 第四步: 在“服务器地址 (Server Address)”一栏中，输入我们的IP: heartsteal.2m.pub
  5. 第五步: 点击“完成 (Done)”，然后在服务器列表中找到HeartSteal，点击“加入服务器 (Join Server)”。
