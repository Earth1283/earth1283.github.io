基础指令

/towny

查看Towny插件基本信息。

/towny ?

查看Towny指令的帮助列表。

/towny map

查看城镇区块地图。

/towny prices

查看建立城镇/国家等的价格。

/towny time

查看新一天到来前所需的时间 (用于收税)。

玩家与好友

/resident

查看自己的玩家信息。

/resident [玩家名]

查看其他玩家的信息。

/resident friend add [玩家名]

添加好友。好友可以拥有你地块的权限。

/resident friend remove [玩家名]

移除好友。

/resident list

查看在线玩家列表。

城镇指令

/town

查看自己所在城镇的信息。

/town new [城镇名]

在你站立的区块上建立一个新城镇。

/town list

查看服务器城镇列表。

/town join [城镇名]

加入一个开放的城镇。

/town leave

离开你所在的城镇。

/town spawn

传送到你的城镇传送点。

/town online

查看镇内在线的玩家。

城镇管理

/town claim

为你的城镇占领脚下的地块。需要城镇银行有足够的资金。

/town unclaim

取消占领脚下的地块。

/town deposit [数量]

向城镇银行存钱。

/town withdraw [数量]

从城镇银行取钱 (仅限镇长/助理)。

/town set mayor [玩家名]

将镇长的职位移交给另一位居民。这是一个不可逆的操作！

/town set taxes [金额]

设置居民每天需要缴纳的税款。税款会自动从居民的余额中扣除并存入城镇银行。

/town set board [信息]

设置城镇公告板信息, 居民上线时会看到。

/town toggle public

切换城镇的开放状态。开放状态下，任何玩家都可以通过 /town join 加入，无需邀请。

地块管理

/plot claim

为自己认领一块可用的地块。

/plot forsale [价格]

将你拥有的地块上架出售。

/plot notforsale

取消出售你的地块。

/plot set shop

将该地块设置为商店地块。需要先放置一个箱子。

/plot set perm [类型] [权限] [on/off]

设置地块权限。这是管理地块访问权限的核心指令。

类型: `resident` (镇民), `ally` (盟友), `outsider` (外人), `friend` (好友)。
权限: `build` (放置方块), `destroy` (破坏方块), `switch` (使用按钮/拉杆), `itemuse` (使用箱子/工作台)。
示例: `/plot set perm friend build on` 允许好友在此地块上放置方块。

/plot toggle [pvp/fire/explosion/mobs]

切换地块上的特定游戏机制。

国家指令

/nation

查看自己所在国家的信息。

/nation new [国家名]

建立一个新国家 (需要是镇长)。

/nation list

查看服务器国家列表。

/nation add [城镇名]

邀请一个城镇加入你的国家。

/nation kick [城镇名]

从你的国家中移除一个城镇。

/nation ally add [国家名]

与其他国家结盟。

/nation enemy add [国家名]

与其他国家宣战。
